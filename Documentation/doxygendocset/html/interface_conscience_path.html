Detailed Description

Facial Features Path Data.

Implementation: Construct from SVG files NSArrays of points, gradients and drawing instructions.

Implementation: Take an outer and dymamic CGPathRef, and populate them with the appropriate drawing instructions.

Object to be populated with the path data from XML.
Path from XML represents an image portion of a feature (iris, teeth, etc.) Possesses points, drawing instructions, style and possible gradientID. Member of ConscienceLayer as a NSMutableArray.

◆ convertToConsciencePath:WithStyle:

- (void) convertToConsciencePath: (NSString *)  pathData
WithStyle: (NSString *)  styleData 

Creates arrays of drawing instructions and coordinates.

Implementation: Using the SVG spec, separate draw instructions from draw points and correlate for future CoreGraphics interpretation.

Source data is direct from xml parsing.

Parameters
pathDataNSString in format of single string of instructions/coordinates
styleDataNSString in format of single string of stroke/fill information

Tokenize path data. Path must be tokenized twice. Once for instructions/coordinates. Twice for X/Y components.
