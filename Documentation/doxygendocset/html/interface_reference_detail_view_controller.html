Detailed Description

Reference information screen.

Implementation: Retrieve ReferenceAsset, ConscienceAsset or Moral from SystemData.

View that is presented when User requests a reference.

Final screen in Reference View Workflow. Lists all relevant information about ReferenceAsset, Moral or ConscienceAsset.
