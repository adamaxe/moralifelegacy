Detailed Description

Implementation: User selects type of ConscienceAsset by tapping on appropriate bounding box surrounding Conscience.

Secondary Accessory selection screen.

View that allows User to select accessories for Conscience.

Second screen in Conscience Accessory Workflow. User can select type of ConscienceAsset via onscreen boxes denoting top, bottom, primary and secondary.
