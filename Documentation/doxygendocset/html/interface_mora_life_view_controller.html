Detailed Description

MoraLife Superclass View Controller.

Implementation: UIViewController Superclass that possesses ModelManager and UserConscience.

Base UIViewController for all MoraLife UIViewControllers.

This UIViewController subclass possesses an init for ModelManager and UserConscience, because every MoraLife UIViewController needs these two items
