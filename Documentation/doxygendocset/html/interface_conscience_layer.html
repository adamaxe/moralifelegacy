Detailed Description

Facial Features Layer.

Implementation: NSMutableArray of ConsciencePaths.

Object to be populated with the Conscience Paths. May contain >1 paths. Layer represents an entire feature (lips, eyelid, etc.) Possesses one or more paths. Member of ConscienceObjectView.
