Detailed Description

Implementation: Conscience Object draws a set of drawing instructions which are pulled from SVG documents.

Conscience vector object drawn with CoreGraphics.

SVG Documents are parsed into arrays of instructions and points. ConscienceObjectView draws via Core Graphics these points when provided the drawing instructions.


Visual Object that represents animatable facial feature, eye, mouth, symbol.

WithInstructions: (NSArray *)  instructions
ForPath: (CGMutablePathRef)  path 

Creates a CGMutablePathRef to draw.

Implementation: To determine which CGPathOperation to perform, each instructions element contains an single instruction.

Parameters
pointsNSArray of (NSNumber *) CGFloats to build path
instructionsNSArray of int values to choose each drawing instruction
pathCGMutablePathRef to be constructed


Depending upon the instruction, a number of points are pulled from points and then added to the path. Path is returned when instructionsArray is empty. Points must be absolute.
