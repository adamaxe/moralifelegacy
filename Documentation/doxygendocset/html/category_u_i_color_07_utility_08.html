Class method for creating a UIColor from a hex formatted string.

Implementation: If alpha is omitted, then assume 1.0 and call other class method.

◆ colorWithHexString:alpha:

+ (UIColor *) colorWithHexString: (NSString *)  hexValue
alpha: (CGFloat)  alpha 

Class method for creating a UIColor from a hex formatted string and alpha value.

Implementation: Split the string into its RGB int components, bitshift each component and divide by 255 to get the 0.0 - 1.0 range required by UIColor.

Parameters
hexValueNSString hex representation to be converted i.e. FF00FF or FF00FF
alphaCGFloat value from 0.0 to 1.0 for alpha channel
Returns
UIColor UIColor from the hexValue and alpha
