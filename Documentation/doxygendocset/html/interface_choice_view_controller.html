Detailed Description

Choice data entry screen.

Implementation: Allows User to enter in Choice data.

Second screen in Moral Entry Workflow. User can either Virtue or Vice depending upon previously selected option.

See also

Commits data to UserData.
Allows access to ChoiceModalViewController to select Virtue/Vice and ChoiceDetailViewController for extended entries. Affects UserConscience by increasing/decreasing mood/enthusiasm.
